```sh
gcc test.c -std=c11 -g test.o
```

`-g` 生成调试信息, 用在gdb中

```sh
gdb a.out

(gdb) l   //查看源码

(gdb)b 4 // 在源码的指定行数放置断点

(gdb) r //运行

```
